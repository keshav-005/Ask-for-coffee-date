# ☕ A Little Virtual Coffee Date

**Live Demo: [ask-for-coffee-date.vercel.app](https://ask-for-coffee-date.vercel.app/)**

Welcome! You probably found this link because I wanted to send you something a little more special than just a text message.

## ✨ Why I Made This
Sometimes, asking someone for coffee—even a virtual one—can be a bit scary. I wanted to turn that invitation into a fun, interactive experience. This little site is my way of saying I appreciate you and would love to spend a bit of time catching up.

## 🎨 How It Works
- **Interactive Invite**: Try saying "No" and see what happens (I promise it's cute!).
- **Animated Scene**: Once you say "Yes," you'll see a cozy little café scene with characters, coffee, and rising steam. 
- **Purely Personal**: No sophisticated servers or databases—just some code and a lot of heart.

## 🛠️ Tech Stack (The Boring Stuff)
- Built with **Vanilla HTML, CSS, and JavaScript**.
- Animations are handled via **CSS Transitions** and **SVG transformations**.
- Fully **responsive** (looks cute on phones too!).
